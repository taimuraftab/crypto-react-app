# CryptoX

CryptoX is a modern cryptocurrency dashboard built with React. It provides real-time market data and interactive charts for popular cryptocurrencies.

## Features

* Live cryptocurrency prices
* 24-hour high and low tracking
* Interactive price charts
* Responsive design for all devices
* Smooth animations using Framer Motion

## Tech Stack

* React
* Tailwind CSS
* Framer Motion
* Recharts
* CoinGecko API

## Installation

1. Clone the repository

```bash
git clone https://github.com/taimuraftab/crypto-react-app.git
```

2. Navigate to the project folder

```bash
cd crypto-react-app
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

## API

This project uses the public CoinGecko API for fetching cryptocurrency data.
